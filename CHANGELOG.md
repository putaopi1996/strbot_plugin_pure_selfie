# 更新日志

## [v6.1.0] - 2025-07-15

### 新增

- WebUI 参考图上传：用 AstrBot 原生 `"type": "file"` 文件上传控件替换了旧的 URL/路径配置方式，直接在浏览器上传参考图。
- 新增 `ignore_keywords` 排除关键词功能：消息包含任一关键词时插件完全不介入，避免与其他插件冲突。
- 上传的参考图以 SHA-256 内容哈希命名持久化到 `uploaded_refs/` 目录，重启不丢失，自动去重。

### 移除

- 移除 `reference_input_mode`、`reference_image_urls`、`reference_image_files`、`reference_image_dir` 配置项。
- 移除 URL 直传参考图和本地目录扫描逻辑，统一走 WebUI 上传 + 本地字节。

### 兼容

- 如果旧配置中仍包含已移除的字段，插件会忽略它们并输出一次弃用警告日志。

## [v6.0.0] - 2026-04-30

### 变更

- 仓库正式收口为"群聊 LLM 自拍专用插件"。
- 主流程只保留：群聊自然语言监听、LLM 判断、预制提示词、多参考图 URL、OpenAI 兼容出图、每群每日次数限制。
- 删除旧的文生图、改图、批量、视频、自拍参考图管理、LLM tools、多 provider 调度及对应服务代码。
- 配置项收缩为 `minimal_selfie` 单一入口，文档与元数据同步更新。

### 清理

- 删除不再对应当前产品形态的历史方案文档和旧功能说明。
- 保留的仓库文档只服务于当前最小自拍版本。

## [v5.0.0] - 2026-04-30

### 说明

- 这是从多功能生图插件向"群聊最小自拍模式"过渡的中间版本。
- 该版本保留了部分兼容层，现已被 `v6.0.0` 完全取代。
